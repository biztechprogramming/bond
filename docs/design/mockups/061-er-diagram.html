📊 Bond Deployment Data Model — ER Diagram

Design Doc 061 — How apps are actually represented in Bond's SpacetimeDB schema

⚠️ The Core Problem

Bond has no apps table. The Deploy tab was incorrectly showing Agents (AI assistants) as applications. The actual deployable units are Components — services, frontends, databases, and systems stored in the components table. Agents operate on deployments; they are not the things being deployed.

❌ Current (Wrong)

Deploy Tab
  └─ Lists AGENTS as "apps"
       └─ Agent name → parsed as app name
       └─ Agent suffix → parsed as environment
       └─ No connection to actual components,
          resources, or environments

✅ Correct

Deploy Tab
  └─ Lists COMPONENTS (deployable units)
       ├─ Each shows ENVIRONMENTS via bindings
       ├─ Each env shows RESOURCES + state
       ├─ ALERTS shown per component/env
       └─ AGENTS available as "operators"
📦 COMPONENTS
PK id string
name, displayName
componentType service|frontend|db|system
FK parentId → self (hierarchy)
runtime node|python|go|rust|java
framework nextjs|fastapi|express|…
repositoryUrl, sourcePath
isActive, discoveredFrom
🌍 ENVIRONMENTS
PK name dev|staging|prod|qa
displayName, order
isActive
maxScriptTimeout
healthCheckInterval
window* maintenance windows
requiredApprovals
🖥️ RESOURCES
PK id string
name, displayName
resourceType server|database|cache|cdn
FK environment → environments
connectionJson host, port, creds
capabilitiesJson cpu, ram, disk
stateJson current snapshot
isActive, lastProbedAt
🔗 COMPONENT_RESOURCES
PK id string
FK componentId → components
FK resourceId → resources
FK environment → environments
port, processName
healthCheck endpoint/command
🔗 RESOURCE_ENVIRONMENTS
PK id string
FK resourceId → resources
FK environmentName → environments
🚨 ALERTS
PK id string
FK environment → environments
FK componentId → components
severity critical|warning|info
message, category
detectedAt, resolvedAt
📏 ALERT_RULES
PK id string
FK environment → environments
FK componentId → components
metric, operator, threshold
severity, enabled
autoFileIssue, triggeredCount
🤖 AGENTS
PK id string
name, displayName
model, utilityModel
systemPrompt, tools
sandboxImage
isActive, isDefault
📡 AGENT_CHANNELS
PK id
FK agentId → agents
channel, enabled
