- **Migrations**: golang-migrate, files in `~/bond/migrations/`, MUST split complex DDL into separate files (FTS5 triggers choke SQLite driver)
- **DB**: SQLite via `aiosqlite`, schema in migrations 000001-000019
- **Backend tests**: `uv run python -m pytest backend/tests/` (pytest-asyncio not installed, async tests ERROR)
- **Frontend**: Next.js + TypeScript, `npm run build` / `tsc --noEmit` for checks
- **API routes**: `backend/app/api/v1/` — settings.py, agent.py, etc.
- **Worker**: `backend/app/worker.py` — main agent loop, context pipeline, LLM calls via litellm
- **Tool selection**: `backend/app/agent/tool_selection.py` — heuristic keyword + momentum, no LLM calls
- **Native tools**: `backend/app/agent/tools/native.py` — file_read (10KB cap), code_execute (stdout truncation)
- **Provider schema**: `providers` → `provider_aliases` + `provider_api_keys` + `llm_models`
- **Vault**: `backend/app/core/vault.py` — get_api_key_with_type(), key_type detection (oauth_token vs api_key)
- **Jobs**: `backend/app/jobs/scheduler.py` + `sync_models.py` — DB-driven, no hardcoded provider registry
- **litellm model IDs**: `{litellm_prefix}/{model_slug}` e.g. `anthropic/claude-opus-4-6`
